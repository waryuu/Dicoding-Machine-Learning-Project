# Dicoding Machine Learning Project

Repositori ini berisi proyek tugas akhir (submission) untuk kelas Machine Learning di [Dicoding Indonesia](https://www.dicoding.com). Proyek ini berfokus pada implementasi alur kerja machine learning mulai dari pemrosesan data, pembuatan model, hingga evaluasi.

## Deskripsi Proyek
Proyek ini bertujuan untuk menyelesaikan tantangan klasifikasi/prediksi menggunakan dataset yang telah ditentukan. Fokus utama dari proyek ini adalah membangun model yang akurat dan efisien menggunakan pustaka populer seperti TensorFlow, Scikit-Learn, atau Keras.

## Fitur Utama
- **Data Preprocessing**: Melakukan pembersihan data, penanganan nilai yang hilang, dan normalisasi.
- **Exploratory Data Analysis (EDA)**: Visualisasi data untuk memahami distribusi dan korelasi antar fitur.
- **Model Development**: Implementasi arsitektur model (misalnya Neural Network atau algoritma Klasifikasi).
- **Evaluation**: Mengukur performa model menggunakan metrik seperti Accuracy, Precision, Recall, atau Loss.

## Teknologi yang Digunakan
- **Bahasa Pemrograman**: Python
- **Library Utama**:
  - TensorFlow / Keras
  - Scikit-learn
  - Pandas & NumPy
  - Matplotlib / Seaborn (untuk visualisasi)
